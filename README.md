## Comandos básicos de Git

| Comando | Qué hace | Cómo se usa |
|---|---|---|
| git --version | Muestra qué versión de Git tienes instalada | git --version |
| git init | Convierte una carpeta normal en un repositorio | git init |
| git add | Marca los archivos que quieres guardar | git add . |
| git commit | Hace el guardado con una descripción | git commit -m "mi primer guardado" |
| git status | Te muestra qué archivos cambiaron | git status |

---

## El orden típico de trabajo

Cada vez que cambias algo en tu proyecto, haces esto:

1. git add .
2. git commit -m "descripción de lo que hiciste"
3. git status para verificar
